**IntakeIQ** is a web-based nutrition tracking application that helps students monitor daily food intake, calculate calories based on grams consumed, and receive personalized AI-powered feedback to support healthier eating habits.

## Features
- User onboarding with personal details and weight goals  
- Food search with autocomplete using a calorie dataset  
- Calorie calculation based on portion size (grams)  
- Daily intake list with total calories  
- AI-generated personalized nutrition summary  
- 7-day intake and summary history using localStorage  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- CSV data processing  
- Google Gemini AI API  
- Browser LocalStorage  

## How It Works
1. Enter personal information and goals  
2. Search and select food items  
3. Input grams consumed to calculate calories  
4. Generate an AI summary for insights and tips  
5. Review previous daily summaries in history
